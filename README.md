# Smart File Organizer


### Install tool in your system
```bash
    uv sync
```

## Making your own changes

### Creating editable running version for testing the changes in your virtual environment

Remember to run this command in main project folder.

```bash
    uv tool install -e .
```

### Deleting editable version from your virtual environment
```bash
    uv tool uninstall sfo
```
